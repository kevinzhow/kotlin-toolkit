    PublicationId [androidJvm]
    typealias PublicationId = String
    A reference uniquely identifying a publication in the reading app.
    For example, a database primary key for a local publication, or a source URL for a remote one.
    We can't use publication.metadata.identifier directly because it might be null or not really unique in the reading app. That's why sometimes we require an ID provided by the app.
